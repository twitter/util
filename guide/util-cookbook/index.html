Util Cookbook¶

util is a collection of Twitter’s core JVM libraries including: concurrency, I/O, caching, stats, time and duration, conversions and so on. It is written in Scala with the intent to be used from Java as well.

This is a cookbook with recipes on how to accomplish many common tasks.

  • Basics
    • Getting the result out of a Try
    • Conversions between Twitter’s Try and Scala’s Try
    • Controlling Time for tests
  • Futures
    • Conversions between Twitter’s Future and Scala’s Future
    • Blocking or synchronous work
    • Future Recursion
    • Limiting concurrency via semaphores and mutexes
    • Caching Futures
  • Input/Output
    • Getting the byte array out of a Buf
    • Getting the String out of a Buf.Utf8
  • Reader
    • Interface
    • Example Usages
  • Usage from Java
    • Scala’s Functions
    • Compatibility APIs

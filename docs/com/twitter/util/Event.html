com.twitter.util

Event

trait Event[+T] extends AnyRef

Events are instantaneous values, defined only at particular instants in time (cf. Var, which are defined at all times). It is possible to view Events as the discrete counterpart to Var's continuous nature.

Events are observed by registering com.twitter.util.Witness Witnesss to which the Event's values are notified.
